# 设置表格边框

```scss
@mixin table-borders($table-class: 'table', $important: true) {
  #{$table-class} {
    border-collapse: separate;
    border-spacing: 0; /* 确保单元格之间没有间距 */

    > thead > tr > th,
    > tbody > tr > th,
    > tfoot > tr > th,
    > thead > tr > td,
    > tbody > tr > td,
    > tfoot > tr > td {
      border: 1px solid var(--grey-3)#{if($important, ' !important', '')}; /* 设置内边框颜色和宽度 */
    }

    /* 移除左侧和顶部的多余边框 */
    thead th + th,
    tbody td + td,
    tfoot th + th {
      border-left: none#{if($important, ' !important', '')};
    }

    thead tr + tr th,
    tbody tr + tr td,
    tfoot tr + tr th {
      border-top: none#{if($important, ' !important', '')};
    }

    /* 保留右侧和底部的外边框 */
    thead th:first-child,
    tbody td:first-child,
    tfoot th:first-child {
      border-left: none#{if($important, ' !important', '')}; /* 移除左侧的外边框 */
    }

    thead th:last-child,
    tbody td:last-child,
    tfoot th:last-child {
      border-right: none#{if($important, ' !important', '')}; /* 移除右侧的外边框 */
    }

    thead tr:first-child th,
    tbody tr:first-child td,
    tfoot tr:first-child th {
      border-top: none#{if($important, ' !important', '')}; /* 移除顶部的外边框 */
    }

    thead tr:last-child th,
    tbody tr:last-child td,
    tfoot tr:last-child th {
      border-bottom: none#{if($important, ' !important', '')}; /* 移除底部的外边框 */
    }
  }
}
```
