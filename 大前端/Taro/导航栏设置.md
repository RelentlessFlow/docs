# Taro 导航栏设置

### 错误用法

```tsx
Taro.setNavigationBarTitle({
     title: type === 'history' ? '历史版本' : '用药计划详情'
});
```

这种用法如果路由跳转后会透传到下一个页面

### 正确用法

```tsx
useTitle(type === 'history' ? '历史版本' : '用药计划详情');

import { isFunction } from '@hera/type-utils';
import { setNavigationBarTitle } from '@tarojs/taro';
import { useLayoutEffect } from 'react';

/** 设置标题 */
export const useTitle = (title: string | (() => string)) => {
    useLayoutEffect(() => {
        const value = isFunction(title) ? title() : title;
        setNavigationBarTitle({
            title: value
        });
    }, []);
};

```
